🎯 尋寶遊戲 - QR Code 列印面板

🖨️ 準備好了嗎？點擊下方「列印」按鈕，或按 Ctrl+P / Cmd+P 列印頁面

📋 使用說明

  1. 列印 QR Code：點擊下方「列印」按鈕或用瀏覽器的列印功能
  2. 裁切 QR Code：將列印好的 QR Code 裁成方形
  3. 貼在現場：在活動現場的各個點位貼上對應的 QR Code
  4. 參與者掃描：參與者用手機掃描 QR Code → 遊戲自動打開 → 查看故事 → 收集印章

💡 提示：使用高質量紙張列印，確保 QR Code 清晰易掃描

🔗 遊戲網址：https://romanceballroom.github.io/sensory/
